================================================================================
                                VISÃO GERAL DO PROJETO
================================================================================

O Brasil Hilário é um blog de humor e entretenimento desenvolvido em PHP com 
MySQL, focado em conteúdo brasileiro. O projeto inclui um sistema completo de 
gestão de conteúdo, anúncios nativos, configurações visuais personalizáveis e 
funcionalidades avançadas de SEO.

================================================================================
                                IMPLEMENTAÇÕES RECENTES
================================================================================

📅 JANEIRO 2025 - SISTEMA DE POSTS ESPECÍFICOS PARA GRUPOS DE ANÚNCIOS

🔧 PROBLEMA IDENTIFICADO:
- Grupos de anúncios apareciam em todos os posts
- Não havia controle granular sobre onde cada grupo era exibido
- Interface de seleção de posts era lenta e pouco prática

✅ SOLUÇÃO IMPLEMENTADA:

1. NOVA ESTRUTURA DE BANCO DE DADOS:
   - Tabela `grupos_anuncios_posts` para associar grupos a posts específicos
   - Campo `posts_especificos` na tabela `grupos_anuncios`
   - Campo `aparecer_inicio` na tabela `grupos_anuncios`
   - Índices de performance para consultas rápidas

2. BACKEND ATUALIZADO:
   - Métodos novos no `GruposAnunciosManager.php`:
     * `getPostsDoGrupo()` - Buscar posts de um grupo
     * `getAllPosts()` - Buscar todos os posts disponíveis
     * `associarPostsAoGrupo()` - Associar posts a um grupo
     * `removerPostsDoGrupo()` - Remover associações
     * `atualizarConfiguracoesPosts()` - Atualizar configurações
     * `debugGrupo()` - Debug de configurações
     * `debugGrupoParaPost()` - Teste de filtro

3. FRONTEND MELHORADO:
   - Interface de seleção de posts com busca
   - Controles rápidos (Selecionar Todos, Limpar, Últimos 10)
   - Contador em tempo real de posts selecionados
   - Lista com scroll para melhor usabilidade
   - Validação JavaScript melhorada

4. SISTEMA DE FILTROS:
   - Filtro por posts específicos
   - Filtro por página inicial
   - Detecção robusta de página inicial vs post específico
   - Compatibilidade com paginação

5. FERRAMENTAS DE DEBUG:
   - Página `debug-grupos.php` para troubleshooting
   - Teste interativo de filtros
   - Visualização completa de configurações
   - Logs de debug para identificação de problemas

6. CORREÇÕES DE BUGS:
   - Corrigido problema de validação que exigia checkboxes extras
   - Corrigida detecção de página inicial
   - Melhorada lógica de filtros
