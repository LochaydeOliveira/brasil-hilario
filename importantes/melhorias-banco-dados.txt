MELHORIAS NO BANCO DE DADOS - BLOG OTIMIZADO PARA ADSENSE
=====================================================

1. ALTERAÇÕES NA TABELA POSTS
----------------------------
- Adicionada coluna 'visualizacoes' para rastrear visualizações de posts
- Útil para identificar conteúdo mais popular e otimizar posicionamento de anúncios

2. NOVAS TABELAS
---------------
a) Tags e Posts_Tags
   - Sistema de tags para melhor organização do conteúdo
   - Relacionamento muitos-para-muitos entre posts e tags
   - Melhora SEO e navegação do site

b) Comentários
   - Sistema de comentários para aumentar engajamento
   - Campos: nome, email, comentário, aprovação
   - Importante para SEO e interação com leitores

c) AdSense Metrics
   - Rastreamento de métricas do AdSense por post
   - Campos: impressões, cliques, receita, data
   - Ajuda a otimizar monetização

d) Configurações
   - Armazenamento flexível de configurações do site
   - Suporte a diferentes tipos de dados (string, integer, boolean, json)
   - Facilita manutenção e personalização

e) Newsletter
   - Sistema de captura de emails
   - Confirmação por token
   - Importante para marketing e retenção de leitores

f) Métricas de Performance
   - Rastreamento de performance por post
   - Campos: tempo de carregamento, tamanho da página, taxa de rejeição
   - Crucial para otimização de anúncios

3. MELHORIAS NA TABELA USUÁRIOS
------------------------------
- Adicionado campo 'tipo' (admin, editor, autor)
- Campo para avatar
- Campo para biografia
- Rastreamento de último login
- Status do usuário (ativo, inativo, bloqueado)
